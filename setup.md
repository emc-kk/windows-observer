# セットアップ手順
## 必要なもの
- Windows 10/11
  - ※快適な作業、トラブルシューティングのため4つのUSBがあるものを推奨(マウス、キーボード、USB, USB-CECアダプター)
- USB-CECアダブター
  - 付属のHDMI™ケーブル
  - 付属のUSBケーブル
- CEC対応テレビ
- HDMIケーブル
- windows-observerが入ったUSB

## 1. WindowsとUSB-CECアダプターとテレビを接続する

USB-CECアダプターの表記を確認しつつそれぞれ接続
- DATA: 付属のUSBケーブルでwindowsと接続
- PC  : 付属のHDMIケーブルでwindowsと接続
- TV  : HDMIケーブルでテレビと接続

<img src="https://github.com/user-attachments/assets/ea07accd-411a-49cd-81c5-aeecafd7fee1" width="500px">

## 2.　USBを接続

windows-observerが入ったUSBをwindowsに接続してください

## 3. Setupの実行

windows-observerフォルダをUSBフォルダから開いて、Setup.batを右クリックして管理者権限で実行してください
※管理者権限で実行が出てこない場合は一度Setup.batをダブルクリックして権限で弾かれてから再度確認してみてください

<img src="https://github.com/user-attachments/assets/da41a907-dd8b-44d1-9463-aab2098baa96" width="500px">

## 4. 動作確認
セットアップが終わったらエンドポイントから動作確認可能です

http://localhost:8765/tv-state
